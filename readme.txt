Use Ctrl+S to save and preview a post!

== Description ==

This plugin allows you to quickly save and preview your post using Ctrl+S (Cmd+S). You can configure the plugin to open a preview in a new window or the current window. 

== Installation ==

1. Upload the `quick-preview` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
